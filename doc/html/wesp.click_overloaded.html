wesp.click_overloaded module¶

This module contains overloads of Click classes to allow custom reactions. The class CustomGroup overloads the class click.core.Group to allow custom parsing and rearranging of the options.

The class OnlyRequiredIf overloads the class click.core.Option to allow an option to be required if a certain SNMP version has been set.

The class CommandAllowConfigFile overloads the class click.core.Command to allow a command to access the configfile.

class wesp.click_overloaded.CustomGroup(name=None, commands=None, **attrs)[source]¶

Bases: click.core.Group

This class overloads the class click.core.Group. It will ensure that the config file is loaded before any other parameter is evaluated and the required options do not suppress the help option. In addition the parameter SNMP version is moved to the front of the args list to enable the class OnlyRequiredIf to set an option (e.g. community) only to required if the corresponding version is set (e.g. 2c).

class wesp.click_overloaded.OnlyRequiredIf(*args, **kwargs)[source]¶

Bases: click.core.Option

This class overloads Option It enables the use of the only_required_if_version attribute This will ensure that that the option with this attribute is only required if the given version is presented. If version and option does not match (e.g. version 3 and a community string) an error is raised.
