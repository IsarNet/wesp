wesp.snmp module¶

This module contains everything related to the SNMP protocol. Every communication with the WLC will run through the Snmp class. For more information see Snmp

The project has been developed and tested using a Cisco 2500er series WLC (SW version 8.3.130.0) and Cisco Aironet 802 11n Draft 2.0 Dual Band APs.

class wesp.snmp.Snmp(ctx)[source]¶

This class handles all interactions via SNMP with the WLC. It provides the basics functions to get or walk with SNMP. The SNMP session will be initialized in the init function. All functions are static to allow easy use without a reference to specific sessions.

static get_by_mac_address(mac_address, separator=':')[source]¶

will return in the information which is stored at the given OID and is specific by the given MAC Address. It will append the MAC Address in decimal format to the given OID and GET this.

Parameters:

  • mac_address – MAC Address of client
  • separator – optional separator to split the mac address. Default ‘:’
Returns:

value stored at this OID. Type depends on value.
